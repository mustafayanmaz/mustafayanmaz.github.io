Hakkımda

Merhaba, ben Mustafa Yanmaz. Recep Tayyip Erdoğan Üniversitesi'nde Bilgisayar Mühendisliği 3. sınıf öğrencisiyim. Web ve mobil uygulama geliştirmeye ilgi duyuyorum.

Hakkımda Detaylar

Bilgisayar mühendisliği eğitimi alarak teknolojinin farklı alanlarını keşfetmeye çalışıyorum. Özellikle web ve mobil uygulama geliştirme üzerine yoğunlaşıyorum.

Modern web teknolojilerini (HTML, CSS, JavaScript) kullanarak kullanıcı dostu ve performanslı web uygulamaları geliştiriyorum.
Android platformu için Java kullanarak mobil uygulamalar geliştiriyorum. Yapay zeka teknolojilerine özel ilgi duyuyor ve bu alanda kendimi geliştirmeye yönelik çalışmalarda bulunuyorum.

Üniversite eğitiminin yanı sıra farklı yarışmalar için projeler oluşturmaya çalışıyorum.

Becerilerim



  • Java ile Android geliştirme

  • Yapay zeka ve makine öğrenmesi

  • HTML, CSS, JavaScript

  • MySQL

İlgi Alanlarım

Yapay zeka, mobil geliştirme ve web teknolojileri üzerine yoğunlaşıyorum.Yazılım geliştirme dışında, yeni teknolojiler ve yapay zeka alanlarına ilgi duyuyorum. Boş zamanlarımda farklı yazılım dillerini ve frameworkleri öğrenerek kendimi geliştirmeye çalışıyorum.
